# odin-recipes
This project is part of The Odin project fondations course.

It is a simple HTML website that contains:
- A main index page
- Three recipe pages
- Basic HTML structure and links between pages

## Live Preview
https://mohamedbouabdallah.github.io/odin-recipes/
